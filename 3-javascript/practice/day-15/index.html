Day 15 (Modules et Promesses en Javascript)

Exercices sur les modules en JavaScript

  1. Module Export/Import : Créez deux fichiers JavaScript, l'un avec une fonction add qui additionne deux nombres, et l'autre qui importe cette fonction et l'utilise pour ajouter deux nombres. Assurez-vous que cela fonctionne correctement.
  2. Module de Calcul : Créez un module JavaScript contenant plusieurs fonctions mathématiques telles que l'addition, la soustraction, la multiplication et la division. Importez ce module dans un autre fichier JavaScript et utilisez ces fonctions pour effectuer divers calculs.
  3. Module de Liste : Créez un module qui expose une liste de noms de fruits. Importez ce module dans un autre fichier JavaScript et utilisez-le pour afficher la liste des fruits dans la console.
  4. Module de Configuration : Créez un module qui contient des constantes de configuration, par exemple, un module de configuration pour une application de calculatrice. Importez ce module et utilisez les constantes de configuration dans votre application.

Exercices sur les promesses en JavaScript

  1. Promesse Simple : Créez une promesse qui résout après une courte attente (par exemple, 2 secondes) avec un message de succès. Ensuite, utilisez .then() pour afficher ce message lorsque la promesse est résolue.
  2. Promesse Rejetée : Créez une promesse qui est rejetée après un court délai avec un message d'erreur. Utilisez .catch() pour gérer le rejet de la promesse et afficher le message d'erreur.
  3. Promesses en Parallèle : Créez trois promesses distinctes qui effectuent des tâches différentes. Utilisez Promise.all() pour attendre que toutes les promesses se terminent, puis affichez un message une fois qu'elles sont toutes résolues.
  4. Promesse avec Fonction Asynchrone : Créez une fonction asynchrone qui retourne une promesse. À l'intérieur de la fonction, effectuez une opération asynchrone (par exemple, un délai avec setTimeout) avant de résoudre la promesse avec un résultat. Appelez cette fonction et utilisez await pour obtenir le résultat de la promesse.
