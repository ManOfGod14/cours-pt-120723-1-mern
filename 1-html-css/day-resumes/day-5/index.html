Les positions en CSS

La position static signifie que l'élément est positionné selon le flux normal du document. il n'est pas affecté par les propriétés de positionnement : top, right, bottom ou left
La position relative : l'élément est positionné par rapport à sa position d'origine dans le flux de contenu. Il est possible d'utiliser les propriétés top, right, bottom ou left pour déplacer mon élément de sa position.
Contenu textuel du parent
La position absolue : l'élément est positionné par rapport à son parent le plus proche qui a une position relative, absolue ou fixe. Si aucun parent n'est trouvé avec ces positions, l'élément est positionné par rapport au corps de la page (le viewport)
La position fixe : l'élement est positionné par rapport au viewport (fenêtre d'affichage) et reste fixe même si la page est défilée.
La position sticky : l'élément est positionné comme relative, jusqu'à ce que l'utilisateur fasse défiler la page jusqu'à ce que l'élement atteigne une position spécifiée et devient fixe.
